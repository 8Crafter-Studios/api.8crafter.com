When a vex is idle, it wanders, selecting air blocks from within a 15×11×15 needs testing cuboid range centered at BoundX, BoundY, BoundZ. when it summoned the vex, this value is set to true, and the central spot is the location of the evoker. Or if an evoker was not involved, this value is false.
